# Express Server (TypeScript Minimal)

Minimal Express server implemented in TypeScript.

Quick start:

```bash
cd /path/to/express-server-fast
npm install
npm run build
npm start
```

Endpoints:
- GET / -> { message }
- GET /health -> { status: 'ok' }
- POST /echo -> echoes JSON body

Development (fast reload):

```bash
npm run dev
```
